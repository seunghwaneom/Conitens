# Conitens Execution Layer: Vibe-kit 기반 Agent Dev System

## Overview
Vibe-kit은 agent execution layer를 담당한다.

## Core Features
- Indexer
- Context DB (SQLite FTS)
- Watcher
- Summarizer (LATEST_CONTEXT)
- Precommit validation

## Workflow
1. Read context
2. Analyze impact
3. Modify code
4. Validate
